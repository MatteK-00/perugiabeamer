perugiabeamer
=============

Inizio documento Latex

```
\documentclass{beamer}
\usepackage{perugiabeamer}

\title{titolo tesi}
\author{Candidato:  \\
		    Relatore:  \bigskip}
\date{}

\begin{document}
	\frametitle{}
	\maketitle

\end{document}

```

Ogni slide viene scritta come segue:
```
\begin{frame}
\frametitle{titolo slide}


\end{frame}
```
